# spotify-data-analysis-dashboard
Interactive Power BI dashboard exploring Spotify music industry trends. Analyzes release growth, explicit content rise, audio features impact on popularity, Top 10 artist comparisons, and detailed drill-through analysis using star schema and advanced DAX calculations.

## Functionality
- Data analysis based on Spotify dataset from Kaggle[https://www.kaggle.com/datasets/tonygordonjr/spotify-dataset-2023]
- Interactive charts and slicers (Year, Genre, Artist, Track)
- KPI indicators with dynamic filtering
- Top 10 artist comparison (Selected vs Average)
- Drill-through artist detail analysis
- Reports and advanced visualization

## Required Software
- Power BI Desktop
- Kaggle Spotify dataset (CSV files)


## How to Use
1. Download the Spotify_Music_Trends.pbix file
2. Open it in Power BI Desktop
3. Use filters and slicers to explore the data
4. Navigate through bookmarks and drill-through pages
